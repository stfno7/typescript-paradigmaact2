#Proyecto TypeScript - Paradigma de Programación (Actividad N° 2)
## Descripción
Proyecto básico desarrollado en TypeScript con el objetivo de analizar el lenguaje, identificar su paradigma y presentar un ejemplo práctico.

## Objetivo
Demostrar el uso del paradigma de Programación Orientada a Objetos (POO) mediante un ejemplo simple, utilizando clases, encapsulación y métodos.

## Requisitos
Node.js instalado

Ingresar en una terminal desde un editor de codigo

git clone https://github.com/stfno7/typescript-paradigmaact2

cd typescript-paradigmaact2

npm install

npx tsc

node dist/index.js
